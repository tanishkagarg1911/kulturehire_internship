# 📊 Gen-Z Career Aspirations Analysis

## 📌 Overview

This project analyzes the career aspirations, influencing factors, and decision-making patterns of Gen-Z using survey data. The goal is to extract meaningful insights about how Gen-Z chooses careers based on financial goals, personal values, and social impact.

---

## 🎯 Objectives

* Identify the most preferred career paths among Gen-Z
* Analyze key factors influencing career choices
* Understand the impact of financial goals (salary, benefits)
* Evaluate the role of personal values and social impact
* Perform demographic analysis (pincode, gender distribution)

---

## 🗂️ Dataset Description

The dataset contains survey responses with the following key fields:

* Pincode
* Gender
* Career preferences (multiple columns)
* Influencing factors
* Work-related perception (social impact, mission alignment)

---

## 🛠️ Tools & Technologies

* **Excel** (Data Cleaning, Pivot Tables, Dashboard)
* **SQL (MySQL)** (Data Analysis, Aggregations)
* **CSV Handling** (Data Import)

---

## 📊 Key Analysis Performed

### 1. Career Preferences Analysis

* Combined multiple career columns using UNION
* Identified top preferred career paths
* Ranked industries based on frequency

---

### 2. Influencing Factors

* Aggregated multiple factor columns
* Identified top drivers:

  * Career growth
  * Salary
  * Work-life balance

---

### 3. Financial Impact Analysis

* Segmented respondents based on salary/benefits preference
* Observed higher inclination toward:

  * Data Analytics
  * Software Development
  * Business Operations

---

### 4. Personal Values & Social Impact

* Analyzed responses related to:

  * Lack of social impact
  * Mission misalignment
  * Unclear purpose

* Found that Gen-Z prefers:

  * Purpose-driven roles
  * Meaningful work environments

---

### 5. Demographic Analysis

* Gender distribution across pincodes
* Used Pivot Tables and COUNTIFS for aggregation

---

## 📈 Dashboard Features

* Interactive slicers (Gender, Pincode, etc.)
* Clean visualizations for:

  * Career trends
  * Factor distribution
  * Demographic breakdown

---

## 🧠 Key Insights

* Gen-Z prioritizes **growth + salary + work-life balance**
* Strong shift toward **data and tech-driven careers**
* Increasing importance of **purpose and social impact**
* Clear segmentation:

  * Financial-driven
  * Values-driven
  * Balanced group

---

## 💼 Conclusion

Gen-Z career decisions are influenced by a combination of financial goals, personal values, and lifestyle preferences. While high-paying roles remain attractive, there is a noticeable shift toward meaningful and purpose-driven careers.
